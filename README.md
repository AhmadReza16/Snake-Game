# 🐍 Snake Game

A classic **Snake Game** built entirely in **JavaScript** for the browser.


## 🛠️ Built With

- **HTML5** – page structure  
- **CSS3** – simple styling  
- **JavaScript (ES6)** – game logic & rendering with the `<canvas>` API
